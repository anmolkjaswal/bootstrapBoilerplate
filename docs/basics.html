Basics of the Internet

  • The internet is essentially a large collection of networks which information to be passed on to other devices around the world
  • The world wide web consists of documents called web pages that are interconnected and accessible via the internet
  • A web page is a single document that can be accessed through the internet & is commonly written in HTML
  • A browser is a software program that allows users to retrieve data from the internet
  • URL - Uniform Resource Locator - This is the address of a particular website
